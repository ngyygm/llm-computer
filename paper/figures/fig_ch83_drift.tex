  The current agent ecosystem (right) suffers from four sources of drift: model version drift
  (output format changes silently across updates), tool schema drift (external APIs change without warning),
  protocol drift (MCP and A2A specifications are still evolving), and skill-pack drift
  (reusable skills behave differently across model versions).
  Within ICA, interface drift primarily affects L4 (semantic interface) and the L4--L5 boundary.}
